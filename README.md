# Karabiner Config

Personal **Karabiner-Elements configuration** for macOS built around a **Hyper key layer**.

Hyper key = `⌃ Control + ⌥ Option + ⇧ Shift + ⌘ Command`

**Caps Lock**
- Hold → Hyper key
- Tap → Backspace

---

# Complex Modifications

**Disable sysdiagnose shortcut**  
Prevents accidental `Ctrl + Opt + Cmd + Shift + .`.

**Cmd+Q double press**  
Requires pressing `Cmd + Q` twice to quit an app.

---

## Navigation

**Hyper + WASD**  
Arrow key navigation.

**Hyper + HJKL**  
Vim-style arrow navigation.

---

## Editing

**Hyper + Tab** → Escape  
**Hyper + E** → Enter  
**Hyper + 2** → Backspace  
**Hyper + [** → Backspace  
**= + Delete** → Forward delete

---

## Application Shortcuts

**Hyper + F**  
Activate Shortcat.

**Hyper + T**  
Open PyCharm terminal.

**Hyper + R**  
Run file in PyCharm.

---

## Utilities

**Ctrl + Shift + D**  
Open `~/Downloads`.
